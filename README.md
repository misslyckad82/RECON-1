# script1.sh

Detta script används för att göra en kontroll av Linux-miljöer. 

Kommandot top visar realtidsövervakning av systemet, främst CPU, minne och processer.

Tryck på följande tangenter medan top kör:
p sortera efter CPU
M sortera efter minne
T sortera efter körtid
k döda process (ange PID) För att avbryta kill-prompten tryck crtl+C
1 visa alla CPU-kärnor
h hjälp
c visa hela kommandoraden
u visa processer för viss användare
q avsluta

FÖR ATT AVSLUTA HELA KÖRNINGEN TRYCK CTRL+C
